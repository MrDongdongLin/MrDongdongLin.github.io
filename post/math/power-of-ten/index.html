Power of Ten

2017-11-15 00:00

问题：如何计算$10^n$，并求其二进制表示的值包含多少个1-bit？

常规做法

import math
p = math.pow(10, n) # 计算10^n
print sum(map(int, bin(int(p))[2:])) # bin函数要求输入整型参数，故此用到了作int转换

使用type函数查看pow函数返回值的类型，发现是float型的，这会带来一些问题，后面会讨论。

像计算机一样思考

计算机的思维模式就是0-1模式，如果能将计算$10^n$转化为binary模式那就最好不过了。通过分解$10^n$可以发现

$$10^n = 2^n \cdot 5^n = 2^n \cdot (2^2+1)^n,$$

将二项式展开，得到

$$10^n = \sum_{i=0}^n C_n^i \cdot 2^{3n-2i}.$$

很好，出现了2^m这样的形式，就说明可以用移位运算了，即使不了解计算机组成原理的人都知道，在计算机里，移位运算比乘法运算要快得多。接下来我们只要计算出二项式系数$C_n^i$即可。

计算二项式系数$C_n^i$

数学上，二项式系数 $$ C_n^i = \frac{n!}{i!\cdot(n-i)!}. $$ 随着$n$的增大，阶乘的增长趋势高于乘方，因此不能用递归、迭代求阶乘。通过查阅资料，我们发现求二项式系数$ C_n^i $可以用动态规划来解决，那么求$10^n$剩下的就是移位和累加操作了。可以通过参阅下面的代码体会上述想法。

def bicoeff(n, i):
    C=cl.defaultdict(int)
    for row in xrange(n+1):
        C[row,0]=1
        for col in xrange(1,i+1):
            if col <= row:
                C[row,col]=C[row-1,col-1]+C[row-1,col]
    return C[n, i]

def PowerOfTen(n):
    res = 0
    for i in xrange(n+1):
        bico = bicoeff(n, i)
        res = (bico << (3*n - 2*i)) + res
    return res

通过type查看返回值类型，发现是long型的。

返回值类型

姑且将pow函数的计算方法称为方法A，将上一节的方法称为方法B，本节将探究一下返回值类型对结果的影响。 先令n=20，计算$10^n$，再使用int强制转换，两种方法的结果为

10000000000
10000000000

似乎没有什么影响啊！二者转二进制求1-bit数量的结果是一致的。 试着增大n，令n=20，结果为

100000000000000000000
100000000000000000000

还是一致，再增大，令n=30，结果为

1000000000000000000000000000000
1000000000000000019884624838656

可以看到，这次的结果居然不一致！我猜测是浮点数的影响，今天太晚了，都凌晨两点多了，这里留个TODO。 总之，使用方法B可以得到正确的结果，通过两幅图我们可以清晰地观察到$10^n$的二进制形式中1-bit数量的增长趋势，图中横坐标表示指数$n$，纵坐标表示1-bit数量。
