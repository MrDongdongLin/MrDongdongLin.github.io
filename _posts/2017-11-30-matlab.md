# 基础篇

- 变量

```matlab
a = 1;
b = 2.3;
c = a+b;
```

- 结构体

```matlab
s.a = 1;
s.b = 2;
s.c = 3;
```

- 函数

```matlab
function [a, b] = func(c, d, e)
% 函数体
end
```

下面介绍一种更灵活的写法，结合结构体能写出让人更加易懂的代码。

```matlab
function varargout = func(varargin)
% 参数检查，若输入参数少于一个或者多余三个，就报错。
nargchk(1, 3);
% 输入参数初始化。
if varargin == 1
    a = varargin{1};
elseif varargin == 2
    a = varargin{1};
    b = varargin{2};
else
    a = varargin{1};
    b = varargin{2};
    c = varargin{3};
end
```
