## 插入图片以及添加说明文字

如果论文中插入的图片乱跑，在主体内容中形成**大段的空白**怎么办？这一点是不能容忍的。一般来说可以通过调整图片的位置来解决这个问题，那万一图片和对应的caption不在同一页又该怎么办？通过查阅资料，我参考了[知乎的一个回答](https://www.zhihu.com/question/30521304/answer/48445567)来解决这个问题。

解决办法：添加一个两行一列的表格 -> 将图片和对应的caption加进去 -> 设置表格属性 -> 行 -> 去掉勾选允许跨页断行 -> 设置表格为无边框。

![view](/img/thesis/figure.png)

## 公式排版

- **从tex源码编辑**

由于我之前是用latex进行排版的，文字可以复制到word中，那公式怎么办？如果你的word安装了最新版的mathtype，这一点就不难解决。首先将tex文件中的内容复制到word中，然后全选，点击mathtype -> Toggle Tex 一键将tex公式源码转化为公式。值得注意的是，mathtype并不能完全识别tex源码，这里可能还需要费点体力，不过相比自己用mathtype直接打公式，这种方式不知道快了多少倍。

![view](/img/thesis/eq1.png)
![view](/img/thesis/eq2.png)

- **带编号的公式及其引用**

特别提醒：<font color=#FF0000>a. 不是所有公式都要带编号！需要在文中引用的公式才需要带编号。b. 公式是句子的一部分！！不要用冒号引出公式！</font>这两点极为重要，很多人都会犯错，甚至检查的人也不知道要这么做……

带编号的公式可以用mathtype添加，在你的大纲都编辑好的情况下（也就是章节标题设为一级，节标题设为二级等等），打开mathtype -> 勾选chapter number，equation number和enclosure即可。

![view](/img/thesis/引用公式1.png)

在正文中需要引用的地方，点击mathtype -> insert reference -> 文中出现equation reference goes here -> 双击对应公式后的标号即可。

![view](/img/thesis/引用公式2.png)
![view](/img/thesis/引用公式3.png)

## 参考文献

我敢断言，<font color=#FF0000>参考文献没有排好的话，一篇论文的质量也好不到哪儿去</font>。参考文献的选择以及排版都极其重要。这里介绍一下如何用word较为方便地排版参考文献（只能说较为方便，因为它跟latex比起来还是差远了！）

需要的工具：Endnote，你学校对应的style文件，后缀为.ens（我用的是符合GBT7714-2005标准的格式文件"20140616HUST_PhD_thesis.ens"。）

**a. 使用endnote导入ens文件。**

**b. 首先，准备好事先整理的bib文件，通过Endnote导入bib文件中的条目。**

![view](/img/thesis/bib1.png)
![view](/img/thesis/bib2.png)

**c. 光标停留在word中需要插入参考文献的位置**

**d. 在endnote软件中选中对应的参考文献，点击insert citation完成插入**

**e. 对参考文献的格式（比如缩进）进行微调直至完全符合标准。**

## 自定义样式的制作

制作样式有一个好处，就是在大篇幅文本的排版中不需要逐段对格式进行调整，可以节省大量的排版时间，并减少出错的机率。如下图，点击红圈处新样式开始制作。

![view](/img/thesis/style1.png)

## 写在最后

Word排版虽然不如latex来的方便，但是绝不是说word排版不如latex好看。总之在排版过程中要密切注意字体、空格、行间距等细节问题，细节都处理不好的话，论文也写不好。关于写论文需要注意的问题就更多了，不能一言以蔽之。平时要多练习，然后多记笔记，不然一段时间不练可能就忘掉了。
