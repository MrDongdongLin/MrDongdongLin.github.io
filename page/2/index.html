一起来做PPT

发表于 2019-06-18 | 分类于 research
字数统计: 1.9k | 阅读时长 ≈ 6

此文提及的材料由PPT制作大师胡XH倾情提供。

写在前面

想成为像胡XH那样的PPT大师，你可能需要掌握多种工具，比如PPT, Visio, Adobe Illustrator, 甚至PS。当然，如果不是为了像胡大师靠开设讲座发家致富，咱没必要精通这么多工具，稍微会一点就ok了。

本文将以胡大师的PPT为例，带领大家体验做PPT的美妙。

前期准备

工欲善其事，必先利其器，首先你得

  1. 安装Microsoft Power Point，不低于2010版就完事了。我装的是最新2019版，使用OTP破解就行，网上破解工具多的亚批，可自行百度关键词office 2019 OTP 破解。
  2. 安装islide，这是一款矢量图插件，你需要用它画图，这样就不需要学illustrator了，可真棒呢。
  3. 安装mathtype，PPT本身也可以输入公式，但是没有mathype渲染的好看。不过mathtype公式大小和位置需要调整，也挺麻烦的，各位可以自行权衡。
  4. 装好软件后，新建一个PPT，打开“导航栏->视图->勾选标尺和网格线”，这是为了让大家能够更好地对准单元的位置。
  5. 如果你想用WPS也可以，但是使用不需要破解的东西总让我觉得好像没装到逼，浑身不自在。

封面和目录的处理

  1. 封面由三个基本部分构成：标题，演讲人相关信息和点缀物体。在制作封面的时候，要以标题为中心，一手抓演讲人信息陈述，一手抓背景装饰。
  2. 目录乃至后续内容风格要和封面保持一致，切不可花里胡哨。
  3. 目录要自成一体，狠抓演讲要点。切不可使用诸如“背景介绍”，“实验结果”等过于笼统的标题，而应是“XXX背景介绍”，“XXX实验结果”等。

且看下方胡大师的作品。

  1. 封面标题突出，周围背景起到点缀作用的同时又不会喧宾夺主，真棒！不过演讲人信息过少，只有一个姓名，如果加上毕业院校，师从何人那就更好了。
  2. 目录完全符合我提到的几点要求，不愧是大师！

内容之背景介绍

  1. 做好目录之后，整个PPT就分好了章节，没有太多必要为每个章节再做一页封面，除非你的PPT长达100多页。
  2. 背景介绍可以用文字铺陈，辅以相关图片作为案例。背景中文字多一些没关系，反正没人看:-)
  3. 可以适当运用文字流，做一些指向性的箭头将背景中的要点关联起来。

且看下方胡大师的作品。

  1. 胡大师使用的是文字流铺陈，但是文字流也并非胡乱铺陈，在文字中提取关键字眼“定义”，“目的”，“工具”可以减少观众的阅读时间。
  2. 背景的发展史，采用时间轴递进，这也是非常典型的一种文字流铺陈做法。如果能将关键字眼加粗可以进一步降低观众的理解成本。
  3. 文字流分为左右两个部分，目的和分类一一对应，看过去非常舒服。
  4. 总之就是666。

内容之图文并茂

  1. 图、文字、公式相辅相成。文字起的是说明性作用，图能表达更丰富的信息，公式便于理解。它们是相辅相成的。
  2. 图片可以有caption也可以没有，要看具体情景。如果图片要表达的意思一目了然，可以不需要caption。
  3. 图片还可以是带有数据的，比如柱状图、饼图可以表示占比，折线图可以表示趋势。这些图可以使用PPT中的插入->图表功能实现。
  4. 动图gif可以非常生动地展现内容给观众，适当的可以使用，但不要过多，不然会显得花哨。
  5. 如果是从他处转来的数据，尽可能使用islide重制，直接截图会导致失真。
  6. 可以使用层级结构，比如树状图来表示层级关系。

且看下方胡大师的作品。

  1. 有图、文字和公式。美中不足的是公式是截图的。
  2. 多图展示运用非常得当，体现了标题“高质量人脸图像应用在多个领域”。
  3. 适当数量的gif图为PPT内容增色不少。
  4. 引用了参考文献的在当页PPT底部注明。这里要提醒大家不要把参考文献全放到最后用一页PPT展示，这样很不好，PPT一页讲完要避免溯源。
  5. 树状结构的表示清晰明了。使用小节或者小小节可以表示层级关系，但是树状图可以表示的深度更深。

内容之表格

  1. 表格可以表示丰富的数据。在实验结果部分常会用到表格。
  2. 表格的要素是表头和内容。表头可以使用深色做底色以示区别，内容可以使用文字或者数据，具体情况具体分析。
  3. 如果有需要，可以添加一些说明性文字，说明表格的数据的来源以及意义。
  4. 如果表格转自别处，尽量重制一遍，不要截图。
  5. PPT里的表格和图都不要编号。

且看下方胡大师的作品。

  1. 标题+说明性文字+表格：内容丰富，可解释性强。
  2. 表格中添加了指向箭头，说明了数据间的关系。箭头的作用在PPT中可以说是举足轻重的。
  3. 表格中的公式使用PPT自带的公式工具编辑而成，不会像截图那样带有不透明底色。

内容之数据关系

PPT内容可以包含多种关系：递进、对比还有罗列等等。

  1. 我们常在包含时间的数据中使用递进关系，比如上面胡大师PPT的背景介绍中就包含了大量的递进关系。
  2. 我们常在新老数据之间使用对比关系。比如想说明现有方法较已有方法取得了什么样的进步时，就可以用对比关系。可以用图片作对比，也可以用文字作对比。
  3. 罗列关系使用范围很广，如果内容很多就可以使用罗列。比如胡大师“图像增强——高质量人脸图像应用在多个领域”中就应用了罗列，当然这里是用的图片罗列。

如何做出巨丑的学术PPT？

字数统计: 1.4k | 阅读时长 ≈ 4

写在前面

一眨眼大家的毕业论文写完了，接下来要准备答辩了。大家开开心心做了PPT，高高兴兴答了辩，美滋滋拿个优秀。——该醒醒了！PPT还没做呢！毕业答辩PPT少不了。本人做PPT的水平可以说很烂，不过在我导师的鞭挞下，可以说有了点进步。接下来我准备反其道行之，以学术PPT为例，教您如何做出巨丑的PPT，让您对自己做出来的PPT感到恶心干呕，从此避开这些雷区。

PS：下面用到的反例有一些是从网上截取的，也就是大家常常在某度上搜到的模板，这里并没有贬低这些模板的意思，毕竟每个人风格不太一样。如有冒犯，欢迎来战。（实际上大部分最丑的都是我自己做的）

为了防止恶心到大家，先打预防针，给个正确教程的链接：

  • https://www.zhihu.com/question/37422718/answer/607404904
  • https://www.zhihu.com/question/22446174/answer/155012501

预设置

实际上，能做好预设的几步，那说明您做PPT的意识还是不错的。

幻灯片大小

考虑到少数机构教学设施的落后，投影的幕布可能容不下您的PPT，使得您的PPT投到了隔壁的墙上，影响您的发挥。我建议幻灯片大小尽量调整为4:3。

幻灯片母版

在幻灯片母版里调整好字体、背景以及logo等。

封面

出来混的，大家都有头有脸，PPT也不例外。PPT封面有三个要素：标题、主讲人和logo之类，或许还可以加上日期。接下来我总结一下怎么做巨丑的PPT封面。

配色

首先是颜色，考虑到少数机构的投影仪巨垃圾，有可能您的PPT颜色实地播放的时候可能比想象中的还淡，所以一般学术PPT会选择底色与文字颜色反差比较明显的风格。如果您想刺瞎评委老师们的双眼，我强烈建议您使用清新淡雅的颜色（任何浅色系都可以）。比如

怎么样？看久了是不是白内障都快复发了呢？或者用暗色系也可以，之前不是流行过快闪吗？

评委老师确实要坐稳，不然都来不及听到我PPT讲了什么呢，嘻嘻。

标题

标题非常重要，应该重点突出，字体要大，尽量加粗。相反，如果您想要挑战评委老师们的视力极限，您可以选择类似上一节那种浅色底浅色字的风格，或者突出其他无关紧要但您觉得非常“好看的”内容。比如

不知道的评委老师还以为您要讲如何“简约”呢，嘻嘻。

目录页

目录是要有的，尤其是PPT内容较多的情况下。这个和标题的注意事项差不多，没什么好说的。

内容

重点来了，怎么做出巨丑的PPT，使您省略答辩的环节，直接不能毕业，从而免却许多麻烦呢？我用我年轻时做的PPT当反例为您讲解吧。

一些反例

  • logo。您可以看出我是多么喜爱我的母校，每一页PPT上都有logo和校训。
  • 侧边导航。您知道我费了多大功夫去做这个导航栏吗？这么酷炫的设计，一定会得到大家的赞赏，嘻嘻。
  • 字体。谨遵学术写作的规则，使用Times New Roman和宋体，我也是乖宝宝呢。

怎么样？是不是煞费苦心？也很有想法？那么别人的评价如何呢？我拿给我导师看，他说这就是一坨屎。如果您的PPT也做成我这样，那么恭喜，您也可以成功恶心到别人啦！嘻嘻。

那么怎么样做才算是一本正经呢？

  • 确保一页PPT只讲一个内容。
  • 每一页要有一个能概括该页内容的标题。
  • 配色不要超过三种颜色。
  • 字体使用黑体等，兼容性好，且不像宋体，黑体不会有锯齿边缘。
  • 导航完全多余，可删之。
  • 每一页的logo没有必要，你仔细看的话，大背景下已经有一个超大logo了。

再来看一页：

  • 图1.1在哪里？（其实在前一页）PPT尽量避免回溯。
  • 大段的文字。PPT是用来辅助您讲述的，如果您要讲的话PPT上都有，那么您站在讲台上的作用是什么？PPT播放员吗？

其他反例

我的PPT还是避开了一些槽点，那么其他问题是什么呢？

  • 没有参考文献！在PPT中引用了参考文献但是没有放在对应的页面中，还是那个问题，PPT要避免翻页溯源。
  • 图表、公式等截图！导致放大出现模糊、颜色突兀的现象。
  • 动画太多太杂！学术PPT最多用一种能快速播放的动画就足够了。
  • 其他待总结

写在最后

更多PPT反例等我以后遇到了再添加到此文中。其实最重要的还是怎么讲的问题，首先要想好怎么讲，是按怎样的一种流程介绍您的工作，然后再做PPT，而不是做好了再去讲PPT。要让观众听了不打瞌睡，还能明白你做了什么，这是项技术活。

主要分两种：JCR分区和中科院分区，JCR分区可在WoS上查，中科院分区我不知道，我也没兴趣查。

h指数

https://en.wikipedia.org/wiki/H-index

h-index ，又称为h指数或h因子（h-factor），是一种评价学术成就的新方法。h代表“高引用次数”（high citations），一名科研人员的h指数是指他至多有h篇论文分别被引用了至少h次。h指数能够比较准确地反映一个人的学术成就。一个人的h指数越高，则表明他的论文影响力越大。例如，某人的h指数是20，这表示他已发表的论文中，每篇被引用了至少20次的论文总共有20篇。
