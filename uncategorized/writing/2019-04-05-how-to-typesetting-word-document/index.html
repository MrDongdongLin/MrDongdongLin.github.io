使用word优雅地排版---学习笔记

发表于 2019-04-05 |
字数统计: 1.8k | 阅读时长 ≈ 6

写在前面

一眨眼，距离我赶硕士毕业论文的时间已经过去一年了。我最开始是使用latex排版的，因为我觉得用latex排出来的公式好看，后来听说学校论文查重用pdf可能会有乱码问题，于是我又用word重排了一遍。最后交由我导师把关，修改了几次。个人觉得我的排版技术得到了飞跃性地提升:-) 然而当时学的许多排版技巧我差不多忘光了-.-b 这次重新打开我的硕士毕业论文，再回忆一下，记点笔记，说不定有用呢。

有兴趣的可以下载我的硕士毕业论文进行审阅，欢迎提出修改意见。此处是下载链接

有关文字段落格式，比如大纲级别，段落前后间距磅数，分页等格式可以直接下载我写好的样式文件然后统一更改（附有gif教程）。此处是下载链接

基本设置

显示网格线以及导航目录

打开word文档 -> 视图 -> 勾选标尺，网格线，导航窗格。

显示空格

文件 -> 选项 -> 显示 -> 显示所有格式标记。

正文设置

调整文字宽度

如下图，四个字和三个字的宽度如何保持一致？选中需要调整的三个字符，点击调整文字宽度为4字符即可



调整空格宽度

通过调整空格宽度来调整下划线长度

添加不透明文本框统一下划线长度

有时候调整空格宽度还是太麻烦了，眼睛也看不准。这时候可以通过添加一个不透明的文本框遮盖过长的下划线使其保持长度一致。


缩小中英文单词之间的间距

当我们的文档出现中英文混合编排的时候，英文单词两边与中文文字的间距有时候显得过宽了。调整方法：打开段落设置 -> 中文版式 -> 去掉勾选自动调整中文与西文之间的间隙。


图片排版

如果论文中插入的图片乱跑，在主体内容中形成大段的空白怎么办？这一点是不能容忍的。一般来说可以通过调整图片的位置来解决这个问题，那万一图片和对应的caption不在同一页又该怎么办？通过查阅资料，我参考了知乎的一个回答来解决这个问题。

解决办法：添加一个两行一列的表格 -> 将图片和对应的caption加进去 -> 设置表格属性 -> 行 -> 去掉勾选允许跨页断行 -> 设置表格为无边框。

公式排版

从tex源码编辑

由于我之前是用latex进行排版的，文字可以复制到word中，那公式怎么办？如果你的word安装了最新版的mathtype，这一点就不难解决。首先将tex文件中的内容复制到word中，然后全选，点击mathtype -> Toggle Tex 一键将tex公式源码转化为公式。值得注意的是，mathtype并不能完全识别tex源码，这里可能还需要费点体力，不过相比自己用mathtype直接打公式，这种方式不知道快了多少倍。


带编号的公式及其引用

特别提醒：a. 不是所有公式都要带编号！需要在文中引用的公式才需要带编号。b. 公式是句子的一部分！！不要用冒号引出公式！这两点极为重要，很多人都会犯错，甚至检查的人也不知道要这么做……

带编号的公式可以用mathtype添加，在你的大纲都编辑好的情况下（也就是章节标题设为一级，节标题设为二级等等），打开mathtype -> 勾选chapter number，equation number和enclosure即可。

在正文中需要引用的地方，点击mathtype -> insert reference -> 文中出现equation reference goes here -> 双击对应公式后的标号即可。


行内公式显示不全

如下图，若行间距为固定磅数，导致行内公式显示不全怎么办？只需将行间距磅数调整为单倍行距即可。



伪代码排版

伪代码的排版因不同人的习惯而异，如下图，我个人喜欢用表格来排。这里的对齐竖线是通过插入形状完成的。


表格排版

表格的排版没什么好说的，比较简单，这里只介绍如何制作斜线表头。

斜线表头

可以在边框中选择斜框线，或者插入直线形状也可以。个人觉得后一种方法比较好，比较好打文字。

参考文献

我敢断言，参考文献没有排好的话，一篇论文的质量也好不到哪儿去。参考文献的选择以及排版都极其重要。这里介绍一下如何用word较为方便地排版参考文献（只能说较为方便，因为它跟latex比起来还是差远了！）

需要的工具：Endnote，你学校对应的style文件，后缀为.ens（我用的是符合GBT7714-2005标准的格式文件”20140616HUST_PhD_thesis.ens”。）

a. 使用endnote导入ens文件。
b. 首先，准备好事先整理的bib文件，通过Endnote导入bib文件中的条目。


c. 光标停留在word中需要插入参考文献的位置
d. 在endnote软件中选中对应的参考文献，点击insert citation完成插入
e. 对参考文献的格式（比如缩进）进行微调直至完全符合标准。

自定义样式的制作

制作样式有一个好处，就是在大篇幅文本的排版中不需要逐段对格式进行调整，可以节省大量的排版时间，并减少出错的机率。如下图，点击红圈处新样式开始制作。

以一级节标题为例，根据学校制定的标准调整行间距磅数，缩进，字体等。具体可在左下角格式下拉框中逐一设置。

目录的制作

目录放在最后生成。在正文内容填充完毕之后，选择引用 -> 插入目录。在插入目录之前先调整格式，如下图所示。

页码的制作

插入页码大家都会，问题是如何让正文的阿拉伯数字页码编号与cover的古罗马数字编号各自从第一页开始？只需在你需要开始新的第一页的页眉或者页脚处将链接至前一页页眉去掉即可。

写在最后

Word排版虽然不如latex来的方便，但是绝不是说word排版不如latex好看。总之在排版过程中要密切注意字体、空格、行间距等细节问题，细节都处理不好的话，论文也写不好。关于写论文需要注意的问题就更多了，不能一言以蔽之。平时要多练习，然后多记笔记，不然一段时间不练可能就忘掉了。
